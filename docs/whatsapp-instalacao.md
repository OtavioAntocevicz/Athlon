# Texto WhatsApp — instalação ATHLON (professor → turma)

Copie e adapte o nome da turma / link do APK.

---

## Mensagem completa (Android + iPhone)

```
ATHLON — como instalar o app

🔗 Site: https://athlonsport.vercel.app

🤖 ANDROID
1) Baixe o arquivo APK que enviei neste grupo
2) Abra o arquivo e toque em Instalar
3) Se pedir, permita “Fontes desconhecidas” / instalar deste arquivo
4) Abra o ATHLON e faça login

(O Play Protect pode avisar porque o app não vem da Play Store — é normal neste momento.)

🍎 IPHONE
No iPhone NÃO existe baixar APK.
1) Abra o link no Safari (não use Chrome)
2) Toque em Compartilhar (quadrado com seta para cima)
3) Toque em “Adicionar à Tela de Início” → Adicionar

Qualquer dúvida, me chama.
```

---

## Só Android (quando enviar o APK)

```
Pessoal, app ATHLON no Android:

1) Baixem o APK anexado
2) Abrir → Instalar (permitir fontes desconhecidas se pedir)
3) Login com o usuário/senha que já usam no site

Atualizações de telas saem sozinhas; só mando APK novo se mudar a instalação do app.
```

---

## Só iPhone

```
Quem usa iPhone: não tem APK.

Abram https://athlonsport.vercel.app no Safari → Compartilhar → “Adicionar à Tela de Início”.

Assim o ATHLON fica na tela inicial como um app.
```
